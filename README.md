# 项目简介 python49

本项目是基于Python的Django框架开发的购物系统，支持WebSocket实时消息和支付宝沙箱支付功能。

## 技术栈

- **编程语言**：Python 3.10
- **数据库**：MySQL 8.0
- **Web框架**：Django 5.0.6
- **前端技术**：HTML5, CSS3, JavaScript, jQuery
- **实时通信**：WebSocket（通过Django Channels实现）
- **支付接口**：集成支付宝沙箱支付

## 功能模块

- **首页**：展示商品列表，支持分类、筛选和搜索。
- **用户系统**：登录、注册，修改密码。
- **商品详情**：详细展示商品信息。
- **用户反馈**：收集用户对商品和服务的反馈。
- **订单管理**：查看历史订单，支付订单，评价商品。

### 核心功能亮点

- **支付测试**：支付宝沙箱支付支持线上支付流程的测试。

## 系统角色

- 用户：浏览商品，购买商品，进行评价。
- 商家：管理商品，查看用户反馈，处理订单。

## 运行环境

- Python 3.10环境
- MySQL 8.0数据库
- 适合在支持Django的Web服务器上运行
